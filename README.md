# AHmeeisSA

مساحة رؤية ووقع ما كان.

ليست معرض أعمال تقليديًا، ولا سيرة ذاتية جامدة، ولا أرشيفًا منفصلًا عن الواقع.

هي بيئة عمل وتكوّن، يحدث فيها العمل أثناء حدوثه، وتبقى آثاره جزءًا من تكوينات لاحقة.

## المبادئ الأساسية

* كل شيء يبدأ كخام.
* كل مخرج يمكن أن يصبح خامًا جديدًا.
* ما يحدث داخل المنصة يبقى داخلها.
* الخصوصية هي الحالة الافتراضية.
* المشاركة تتم بحسب مقصد الزائر لا بحسب بنية ثابتة للصفحات.
* المختصر والتفصيل مستويان مختلفان للرؤية، لا محتويان منفصلان.

## المساحات الأولية

### المساحة المهنية

الغرض:

* الزيارات المهنية.
* ملفات التعريف العملية.
* السيرة الذاتية الديناميكية.
* التحضير للمقابلات.
* دراسة الجهات والقطاعات.

### المساحة الفنية

الغرض:

* تصميم الفيديوهات.
* إدارة التواصل مع العملاء.
* مشاركة الملفات والمقترحات.
* متابعة مراحل التكوين حتى المخرج النهائي.

## الإصدار الأول

V1:

* مساحة مهنية قابلة للمشاركة.
* مشروع فيديو خطوبة.
* مشروع فيديو زواج.
* بنية قابلة للتوسع مستقبلًا دون إعادة البناء.
