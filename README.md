# Ahmeeissa

خريطة تفاعلية للقدرات والعلاقات المهنية وليست سيرة ذاتية تقليدية.

## المحاور
- الهندسة كنمط تفكير
- تطوير الأعمال
- إدارة العمليات
- التدريب ونقل المعرفة
- إدارة الحشود والفعاليات
- القيادة الارتجالية وتحمل المسؤولية
- بناء الأنظمة
الهوية
↓
الهندسة
↓
بناء الأنظمة
↓
العمليات
↓
تطوير الأعمال

والتدريب والحشود والقيادة كمسارات متقاطعة
